Open source algorithms

This tool was developed at Lawrence Berkieley National Lab using a Python backend and web standard HTML, CSS, and JavaScript. The backend relies on the CherryPy server framework, the Jinja2 templating system, wkhtmltopdf - a WebKit rendered to convert HTML to PDF, and the scipy, numpy, and matplotlib Python modules for data manipulation and dynamic generation of figures. The source code will be available for examination or third party use upon completion of a legal review, documentation, and code cleanup effort. For early access, please contact sborgeson <at> berkeley <dot> edu.
