building-data-analysis
======================

Tools for parsing, analysing, and visualizing interval data from buildings, including Smart Meter electricity.

To use:
Install python 2.X (developed using 2.7.5)
+ Numpy (1.7.1)
+ Matplotlib (1.2.0)
+ Scipy (0.12.0)
+ CherryPy (3.2.2)
Install wkhtmltopdf
